**Movie List — Просмотр популярных фильмов**

**Запуск:**
npm i
npm run dev

**Описание:**
Приложения для просмотра всех популярных фильмов с ресурса TMDB,
реализующее фильтрацию по жанрам и сотрировку по рэйтингу по фильмам из списка доступных.

**Технологический стек**
Frontend: React 18 (TypeScript)
Стилизация: Tailwind CSS + UI-кит shadcn/ui
State Management: Zustand
API: TMDB REST API

**Архитектура и реализация**
При разработке я сделал упор на чистоту кода и масштабируемость:

Модульная структура: Проект разделен по принципам, близким к FSD (Feature-Sliced Design). Выделены слои pages, features, widgets и shared. Это упрощает поддержку и навигацию по коду.

Управление состоянием: Вся логика работы с данными (запросы к API, фильтрация по жанрам и сортировка по рейтингу) вынесена в Zustand. Это позволило разгрузить компоненты и избежать "prop drilling".

Адаптивность: Приложение полностью адаптивно. Сначала был проработан Desktop-интерфейс, после чего проведена тщательная оптимизация под мобильные устройства.

UI/UX: Использование компонентов от shadcn/ui позволило добиться современного дизайна и высокой доступности (Accessibility) интерфейса.

**Дальнейшие правки:**

Пагинацию: Подгрузку следующих страниц фильмов при скролле или через переключатель страниц.

Детальную страницу фильма: Более подробную информацию, список актеров и трейлеры.

Оптимизацию: Оптимизацию запросов к API.
